# Discord ↔️ Telegram Bridge Bot

Бот для пересылки сообщений из Discord канала в Telegram с указанием времени отправки.

## 🚀 Возможности

- Отслеживание сообщений в указанном Discord канале
- Пересылка сообщений в личные сообщения Telegram
- Красивое форматирование с указанием автора и времени
- Поддержка вложений (ссылки на файлы)
- Логирование работы бота

## 📋 Требования

- Python 3.8+
- Discord Bot Token ИЛИ Discord Account Token
- Telegram Bot Token

## 🛠️ Установка и настройка

### 1. Клонирование и установка зависимостей

```bash
git clone https://github.com/ttwintypink/discord-telegram-bridge.git
cd discord-telegram-bridge
pip install -r requirements.txt
```

### 2. Выбор способа подключения к Discord

#### Способ 1: Использование Discord бота (рекомендуется)
1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Создайте новое приложение
3. Перейдите в раздел "Bot" и создайте бота
4. Скопируйте токен бота
5. Включите "Message Content Intent" в настройках бота
6. Пригласите бота на сервер с нужными правами

#### Способ 2: Использование вашего Discord аккаунта (Self-bot)
⚠️ **ВНИМАНИЕ:** Этот способ нарушает правила Discord и может привести к блокировке аккаунта!

1. Откройте Discord в браузере
2. Нажмите F12 → вкладка "Network"
3. Обновите страницу и найдите запрос к API
4. В заголовках найдите "Authorization" и скопируйте токен
5. Токен выглядит как: `mfa.XXXXXXXX` или `XXXXXXXXXXXXX`

### 3. Создание Telegram бота

1. Найдите [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте токен бота

### 4. Настройка конфигурации

1. Скопируйте файл `.env.example` в `.env`:
```bash
cp .env.example .env
```

2. Отредактируйте `.env` файл:
```env
DISCORD_TOKEN=ВАШ_DISCORD_ТОКЕН
TELEGRAM_TOKEN=ВАШ_TELEGRAM_ТОКЕН
```

### 5. Настройка параметров

В файлах уже указаны:
- ID Discord сервера: `1460084815209566443`
- ID Discord канала: `1475098820365914183`
- ID Telegram пользователя: `1805647541`

При необходимости измените эти значения.

## 🚀 Запуск

### Для Discord бота:
```bash
python discord_telegram_bot.py
```

### Для Self-bot (с вашим аккаунтом):
```bash
python discord_telegram_selfbot.py
```

### Для продвинутого бота с подпиской:
```bash
python discord_telegram_bot_advanced.py
```

## 🔧 Дополнительные настройки

### Изменение ID сервера, канала или пользователя

Откройте файл `discord_telegram_bot.py` и измените значения:

```python
self.DISCORD_SERVER_ID = 1460084815209566443  # ID вашего Discord сервера
self.DISCORD_CHANNEL_ID = 1475098820365914183  # ID вашего Discord канала
self.TELEGRAM_USER_ID = 1805647541  # ID вашего Telegram пользователя
```

### Как узнать ID

**Discord:**
- Включите режим разработчика в Discord (Настройки → Расширенные → Режим разработчика)
- Правый клик по серверу/каналу → "Копировать ID"

**Telegram:**
- Отправьте сообщение боту [@userinfobot](https://t.me/userinfobot)
- Или боту [@getmyid_bot](https://t.me/getmyid_bot)

## 🐛 Устранение проблем

1. **Бот не отвечает на сообщения:**
   - Проверьте правильность токенов в `.env`
   - Убедитесь, что бот имеет права на чтение сообщений в Discord
   - Проверьте, что бот не заблокирован в Telegram

2. **Ошибки при запуске:**
   - Убедитесь, что установлены все зависимости: `pip install -r requirements.txt`
   - Проверьте версию Python (требуется 3.8+)
